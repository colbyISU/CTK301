WEB DESIGN

The project below was a personal project made for the game VALORANT. The goal was to create a website that allowed users to fully customize their inventory and equip any skin of their choice so they can show their dream collections to their friends. The main goal of this project was to implement a fully working login system which was done so using PHP and MySQL.
